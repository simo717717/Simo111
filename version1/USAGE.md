# 照片管理系统使用说明

## 初始设置

### 1. 首次使用
- 当您首次访问系统时，需要先注册一个账户
- 点击导航栏中的"注册"按钮
- 填写注册表单信息：
  - 名字：您的名字
  - 姓氏：您的姓氏
  - 邮箱：有效的电子邮件地址
  - 用户名：唯一的用户名
  - 密码：至少6位的密码
  - 确认密码：再次输入相同的密码
  - 个人简介：关于自己的简短介绍

### 2. 登录系统
- 注册完成后，点击"登录"按钮
- 输入您注册时使用的用户名和密码
- 登录成功后即可访问系统功能

### 3. 系统功能
- **首页**：查看系统信息
- **图库**：浏览和管理您的照片
- **上传**：上传新的照片到系统
- **个人资料**：查看和编辑个人信息

## 后端配置

### 数据库配置
- 确保MySQL数据库已启动
- 在 `backend/src/main/resources/application.yml` 中配置数据库连接信息：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/photo_management
    username: root  # 您的数据库用户名
    password: 123456  # 您的数据库密码
```

### 运行后端
```bash
cd backend
mvn spring-boot:run
```

### 运行前端
```bash
cd frontend
npm run dev
```

## 常见问题

1. **无法注册/登录**
   - 确保后端服务正在运行
   - 检查网络请求是否有错误

2. **数据库连接错误**
   - 确认MySQL服务已启动
   - 检查数据库配置信息是否正确
   - 确认数据库 `photo_management` 已创建

3. **照片上传失败**
   - 确认登录状态
   - 检查文件格式和大小限制
